Issue of Express scribe
 

After successful installation follow the below steps:
  • First, Log out MT Client and close the interface.
  • Go to the folder where you have installed your SecureMTClient VRS. e.g., C:\Program files\SecureXsoft\VRS MTClient\
  • Open following file in notepad MTClient.exe.config
  • Scroll down and you would see following lines
    appSettings
    add key="WebServerName" value="admin.securexsoft.com" /
    add key="PlayerName" value="C:\Program Files (x86)\NCH Swift Sound\Scribe\scribe.exe"
    appSettings
  • Change the path of scribe.exe to your installed path for express scribe. Normally it would be
  • C:\Program Files\NCH Swift Sound\Scribe\scribe.exe
  • Save the file and close notepad.
